Bluff your way through planning poker

What is Bluff?

Bluff is a free bot that supports your team to make estimations with planning poker technique directly in Slack. You can install the app with "Add to Slack" button above.

Main features

  • Streamlined flow - all interactions happen in one place
  • Supports mentioning individual users, custom user groups, @here, @channel
  • Supports channels shared between different workspaces
  • Gives you live status on participation of all mentioned users
  • A user can place a bid either from a predefined set of options or in a free form
  • A user may reject participating in an estimation
  • Estimation can be stopped at any time
  • After estimation has finished results are displayed per person and median value is computed
  • Can be used in any public channel

Privacy policy

The app can be used in any public channel without a need to be invited. However, it doesn't scan through all the channels in a workspace. The only information the app knows about the channel where the slash command was executed is the channel id.

The app cannot access any messages outside the planning poker workflow.

The app can list the users in a channel or in a custom group. This allows you to use group, @channel and @here mentions. The only information the app gets about the users is the user id and name.
